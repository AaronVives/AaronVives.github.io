Uitleg opdracht strings

JavaScript kan teksten voorstellen met strings. Strings kunnen worden gemaakt met enkele aanhalingstekens, zoals let s1 = "blablabla";, of met de constructor new String(). De typeof laat zien welk type een waarde heeft.
Variabelen kunnen verschillende typen waarden bevatten. vb: leeftijd, intrest, en isGevaarlijk bevatten een getal, een decimaal getal, en een boolean. vandaag bevat een Date.
Strings hebben een .length-property om de lengte op te vragen en een .charAt()-methode om individuele karakters op te vragen.
Hoewel tekst[1] soms wordt gebruikt, is het beter om .charAt() te gebruiken omdat het compatibeler is en het de illusie voorkomt dat een string een array is.
