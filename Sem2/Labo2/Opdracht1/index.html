Wat is de return value van de confirm functie als de gebruiker op een van de buttons klikt?

Als er op "OK" wordt geklikt, retourneert de functie true. Bij het klikken op "CANCEL" retourneert de functie false.

Wat is de return value van de prompt functie als de gebruiker een tekst intypt en op "OK" klikt?

Wanneer de gebruiker een tekst invoert en op "OK" klikt, retourneert de functie de ingevoerde waarde. BV: als "iets" wordt ingevoerd, retourneert de functie ook "iets".

Wat is de return value van de prompt functie als de gebruiker op "CANCEL" klikt?

In het geval dat de gebruiker op "CANCEL" klikt, retourneert de functie null.
